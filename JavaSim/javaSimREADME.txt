This zipfile includes the Java version of VBASim discrete-event simulation, JavaSim. Two files are included, this README and JavaSim.jar, which includes class files and JavaSim source code.

Run Java from the command line by navigating to the directory which contains javaSim.jar with the command

java -cp javaSim.jar org.javasim.examples.<ExampleName>

Valid of values of ExampleName are:

AsianOption
FaxCenter
MG1
MG1Lindley
MTMInfinity
SAN
SANMax
TTFSingle
TTFReplications

Consult a Java tutorial, such as The Java Tutorials from Oracle, to learn how to edit the source code and make changes to the simulations.
